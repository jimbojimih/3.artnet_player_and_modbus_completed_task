Выполненное тестовое задание по отправке ArtNet пакетов на DMX конвертеры и передаче данных для управления релейным модулем. 
Данные в файле "Test_Color.ani" представляют собой последовательность байтов, которые были преобразованы в отдельные пакеты для отправки.
Для тестовой проверки программы были развёрнуты три сервера. Для эмуляции релейного модуля был задействован "mtcpServerEmulator". 
Для эмуляции DMX конвертеров использовались "test_artnet_server1" и "test_artnet_server2".
Для запуска программы artnet_player_and_modbus_sender используйте атрибуты по-умолчанию для классов "modbus_sender" и "artnet_and_modbus_sender".
